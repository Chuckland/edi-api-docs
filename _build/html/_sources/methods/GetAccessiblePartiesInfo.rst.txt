GetAccessiblePartiesInfo
=========================

Метод для получения списка организаций, доступных пользователю.
Имя ресурса: /V1/Parties/GetAccessiblePartiesInfo
HTTP метод: GET
В запросе должен присутствовать HTTP-заголовок Authorization с необходимыми данными для авторизации.
В ответе сервер вернет структуру :doc:`PartiesInfo <../structures/PartiesInfo>`.

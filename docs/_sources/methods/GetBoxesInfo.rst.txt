GetBoxesInfo
=============

Метод для получения списка ящиков, доступных пользователю.

Имя ресурса: **/V1/Boxes/GetBoxesInfo**

HTTP метод: **GET**

В запросе должен присутствовать HTTP-заголовок ``Authorization`` с необходимыми данными для авторизации.
В ответе сервер вернет структуру :doc:`BoxesInfo <../structures/BoxesInfo>`.

BankAccount
============

.. code-block:: c#

	class BankAccount {
		BankAccountNumber: string,
		CorrespondentAccountNumber: string,
		BankId: string,
		BankName: string
	}

	
Банковские реквизиты:

 - BankAccountNumber – расчётный счёт,
 - CorrespondentAccountNumber – корр. счёт,
 - BankId – БИК банка,
 - BankName – наименование банка.

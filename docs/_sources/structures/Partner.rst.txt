Partner
========

.. code-block:: c#

	class Partner {
		PartnerId: string,
		PartnerGln: string,
		PartnerName: string
	}
	
Данные организации, зарегистрированной в системе (используется для передачи информации о контрагентах):

 - PartnerId – уникальный идентификатор организации в системе,
 - PartnerGln – GLN.
 - PartnerName – наименование.

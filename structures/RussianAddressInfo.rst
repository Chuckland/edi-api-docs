RussianAddressInfo
===================

.. code-block:: c#

	class RussianAddressInfo {
		PostalCode: string,
		RegionCode: string,
		District: string,
		City: string,
		Village: string,
		Street: string,
		House: string,
		Flat: string
	}
	
Российский адрес:

 - PostalCode – почтовый индекс,
 - RegionCode – код региона в формате ISO,
 - District – район,
 - City – город,
 - Village – населенный пункт,
 - Street – улица,
 - House – дом,
 - Flat – квартира/офис.
